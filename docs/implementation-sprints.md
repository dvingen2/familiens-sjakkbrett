# Implementasjonssprinter

## Sprint 1: Instant Play + Familievarme

Mål:

- gjøre forsiden til den naturlige startflaten
- gjøre familieprofiler mer personlige
- gi partier en hyggeligere emosjonell tone

Leveranser:

- avatarer og profilfarger
- tydeligere familieprofilkort
- bedre topbar-identitet
- varm parti-feedback på hjem og spillside
- dokumentert retning for neste sprint

Ut av scope:

- full badge-motor
- oppdragssystem
- PWA
- sky-migrering av lokale spill

## Sprint 2: Første belønningssløyfe

Mål:

- gi barna grunn til å komme tilbake

Leveranser:

- første badges
- enkle milepæler
- ukens familieutfordring
- post-game “god innsats” og små feiringer

Avhenger av:

- Sprint 1

## Sprint 3: Hjemskjerm og offline

Mål:

- gjøre appen tilgjengelig som en ekte familieapp på mobil

Leveranser:

- web app manifest
- service worker
- offline app shell
- install prompt
- hjemskjermikoner

Avhenger av:

- Sprint 1

## Sprint 4: Læring som støtte

Mål:

- hjelpe særlig 10-åringen uten å gjøre appen “skolete”

Leveranser:

- hint nivå 1
- dagens oppgave
- enkel forklaring av trusler
- første mini-leksjoner

Avhenger av:

- Sprint 2

## Sprint 5: Sky og familiehistorikk

Mål:

- gi verdi på tvers av enheter uten å øke friksjon i kjernen

Leveranser:

- persister lokalt spill ved innlogging
- online-spill mellom profiler
- historikk mot familiemedlemmer

Avhenger av:

- Sprint 1
- Sprint 3
